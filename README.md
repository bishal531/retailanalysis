## Getting Started

### 1. Clone the Repository
```bash
git clone [repository-url]
cd intern1/project_1_eda_retail_sales
```

### 2. Set Up Environment
```bash
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
pip install -r requirements.txt
```

### 3. Download Datasets
- Download from [Kaggle Retail Sales](https://www.kaggle.com/datasets?search=retail+sales)
- Place in `project_1_eda_retail_sales/data/` folder

### 4. Run Analysis
```bash
python eda_retail_sales.py
```
